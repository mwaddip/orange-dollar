# Orange Dollar Website Style Guide (style_guide.md)

## 1. Design Philosophy

-   Aesthetic: Protocol-grade minimalism (Bitcoin × Research × Modern
    DeFi)
-   Core traits: Deterministic, geometric, symmetric, low-noise UI
-   Avoid: gradients, glow effects, glassmorphism, flashy animations

------------------------------------------------------------------------

## 2. Brand Colors (Locked)

Primary Orange: #F7931A\
Pure White: #FFFFFF\
Soft White: #F5F7FA\
Primary Text (near-black): #1A1A1A\
Background Dark: #0F1113

Use pure white (#FFFFFF) for:
- Page backgrounds (light mode)
- Large empty surfaces
- Documentation pages

Use soft white (#F5F7FA) for:
- Logo internal elements
- Dividers
- Cards
- Subtle UI surfaces
- Icons
- Button text on colored backgrounds

Rules: - Always use EXACT #F7931A for brand consistency - Do not
introduce alternative orange shades - Prefer neutral-heavy UI with
orange as accent

------------------------------------------------------------------------

## 3. Typography

Primary Font: Inter

Fallback Stack: Inter, system-ui, -apple-system, Segoe UI, Roboto,
Helvetica, Arial, sans-serif

Weights: - Headings: 600 (SemiBold) - Body: 400 (Regular) - UI
Labels/Buttons: 500 (Medium)

Tracking: - Headings: letter-spacing: -0.02em - Body: normal tracking

Type Scale: - Hero: 48--64px / 600 - Section Headers: 28--36px / 600 -
Body: 16--18px / 400 - Captions: 13--14px / 400 - Buttons: 15--16px /
500

------------------------------------------------------------------------

## 4. Logo Usage

Logo Identity: Geometric coin + 8-segment ring + equilibrium bar/node

Safe Area: - Minimum padding = inner ring thickness

Do NOT: - Add drop shadows - Add gradients - Stretch or distort - Place
on noisy backgrounds

Approved Backgrounds: - #FFFFFF - #0F1113 - #FAFBFC

------------------------------------------------------------------------

## 5. Layout System

Grid: 8pt spacing system (aligns with 8-segment ring symbolism)

Spacing Scale: - 8px: micro spacing - 16px: UI spacing - 32px: block
spacing - 64px: section spacing - 96--128px: hero spacing

Container Width: - 1100px--1280px max

------------------------------------------------------------------------

## 6. UI Components

### Primary Button

background: #F7931A color: #F5F7FA border-radius: 8px font-weight: 500
padding: 12px 20px

Hover: background: #E68212

### Secondary Button

background: transparent border: 1px solid #E6E8EB color: #1A1A1A

Avoid pill-shaped DeFi-style buttons.

------------------------------------------------------------------------

## 7. Iconography

Style Requirements: - Monoline - Geometric - Minimal stroke - No shading
or gradients

Recommended Icon Sets: - Lucide - Phosphor (thin/regular) - Heroicons
(outline)

Avoid: - 3D icons - Duotone icons - Cartoon illustrations

------------------------------------------------------------------------

## 8. Background & Surfaces

Primary Background: #FFFFFF

Dark Mode Background: #0F1113

Section Separation: border-top: 1px solid #F0F2F5

Avoid: - Gradient backgrounds - Animated backgrounds - Texture overlays

------------------------------------------------------------------------

## 9. Motion & Interaction

Animation Style: - Subtle - Deterministic - Functional

Timing: transition: all 0.15s--0.20s ease-out

Allowed: - Soft hover transitions - Subtle ring rotation (optional,
slow)

Avoid: - Parallax effects - Bounce easing - Particle effects - Scroll
hijacking

------------------------------------------------------------------------

## 10. Content Tone (Brand Alignment)

Voice: - Academic - System-oriented - Research-driven - Minimal hype

Use: - Diagrams - Equilibrium visuals - Clean charts - Grid-aligned
layouts

Avoid: - Meme aesthetics - Over-marketing language - Emoji-heavy UI

------------------------------------------------------------------------

## 11. Dark Mode Guidance

Text: - Primary: #EDEFF2 - Secondary: #AEB4BC

Keep: - Orange (#F7931A) unchanged - Soft White (#F5F7FA) for icon
consistency

------------------------------------------------------------------------

## 12. Brand Differentiation Strategy

Positioning: - Orange Dollar = Algorithmic Stability (not fiat
mimicry) - Visual language = neutral UI + precise orange accents

Accent Usage Priority: 1. CTA buttons 2. Active states 3. Data
visualization (peg bands) 4. Highlights

Target Feel: "Deterministic monetary protocol" NOT "flashy crypto
startup"
